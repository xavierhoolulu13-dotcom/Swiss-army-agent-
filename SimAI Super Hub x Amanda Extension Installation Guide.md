# SimAI Super Hub x Amanda Extension Installation Guide

This extension is designed for compatibility with **Microsoft Edge**, **Mozilla Firefox**, and **Tor Browser**.

## 1. Microsoft Edge Installation
1.  Unzip the `amanda-extension.zip` file.
2.  Open Edge and navigate to `edge://extensions`.
3.  Turn on **Developer mode** in the bottom left corner.
4.  Click **Load unpacked** and select the folder where you unzipped the extension.

## 2. Mozilla Firefox & Tor Browser Installation
1.  Open Firefox or Tor Browser.
2.  Navigate to `about:debugging#/runtime/this-firefox`.
3.  Click **Load Temporary Add-on...**.
4.  Select the `manifest.json` file inside the unzipped extension folder.
    *Note: Temporary add-ons are removed when the browser is restarted. For permanent use, the extension would need to be signed by Mozilla.*

## Features Included
*   **SimAI Super Hub UI**: Modern ChatGPT-like interface.
*   **Amanda Orchestration Kernel**: Integrated chat-based control.
*   **8-Lane Pipeline Simulation**: Real-time visual tracking of Ingestion, Sanitization, etc.
*   **Cross-Browser Manifest**: Manifest V3 for Edge and Gecko-compatible settings for Firefox/Tor.
